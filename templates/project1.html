Wifi and Cafe Finder

This website is very helpful for people who work from home. Visitors can find cafés listed with details such as the number of sockets, available tables, and the café’s location.

Project Overview

January 12, 2024 by Tsering

This project is a web application designed for remote workers and students looking for work-from-home friendly cafés. It helps users easily discover coffee shops with reliable Wi-Fi, available seating, and essential amenities to create a productive work environment outside the home.

Tech Stack

  • Backend: Flask (Python)
  • Frontend: Bootstrap (responsive UI)
  • Database: SQL (SQLite during development)
  • Extras: Jinja templates, simple slugged URLs for SEO-friendly links

Modular Architecture

  • app.py – main application file that defines routes.
  • templates/ – contains HTML files (index.html, about.html, portfolio.html, project1.html, etc.) with reusable layouts (header.html, footer.html, base.html).
  • static/ – contains CSS, JS, and images for styling.
  • .env – stores environment variables.
  • requirements.txt – lists dependencies for deployment.

Challenges faced

  • Template Inheritance: Setting up a clean base.html with header/footer and extending it in multiple pages.
  • Static File Handling: Ensuring CSS, JS, and images load correctly through Flask’s url_for.
  • Data Handling: Managing café details without using a full SQLAlchemy database (opted for dictionaries/JSON for simplicity).

What I Learned

  • How to structure a Flask project for scalability.
  • The importance of modular templates (header.html, footer.html, base.html) for reusability.
  • Handling static files properly with Flask.
  • Using Bootstrap effectively to make a responsive, user-friendly UI.

Future Development

  • Add a search and filter option (by location, Wi-Fi speed, or seating).
  • Implement user reviews & ratings.
  • Switch from dictionary/JSON data to a database (SQLite/PostgreSQL) for dynamic storage.
  • Integrate Google Maps API for café location visualization.

Key Features

  • Café Listings – Find cafés suitable for remote work.
  • Wi-Fi Availability – Shows which cafés provide reliable internet.
  • Socket Count – Number of power outlets available for laptops and devices.
  • Seating Capacity – Number of tables or seats for comfortable working.
  • Location Details – Address, map, and directions to the café.
